# HTML Authentication POC

This project is a simple HTML-only authentication flow.

## Pages Included

- index.html (Login Page)
- register.html
- forgot.html
- reset.html
- dashboard.html

## Redirection Flow

1. Login (index.html) → Dashboard
2. Registration → Login
3. Forgot Password → Reset Password
4. Reset Password → Login
5. Logout → Login

## Technologies Used

- HTML only
